# 玩法介绍

## 项目简介

本项目是一款基于 Unity XR 开发的 VR 叙事解谜游戏 Demo。玩家将进入一个昏暗、压抑的卧室空间，通过观察、拾取、触碰、移动和组合场景中的物体，逐步恢复主角与妻子有关的记忆，并在最后做出关键选择。

游戏主题围绕“记忆、愧疚与选择”展开。场景中的每一个关键物品都不只是解谜道具，也承担着叙事功能。玩家需要在 VR 环境中亲手与这些物品互动，从而一步步接近主角不愿面对的真相。

## 游戏背景

玩家醒来后发现自己身处一个黑暗的卧室。房间看似普通，却处处透露出不自然的氛围：电视屏幕上的文字、被锁住的抽屉、散落的衣物、隐藏的戒指、被丢弃的餐厅邀请函，以及妻子的死亡通知单。

随着玩家完成一个个任务，房间中的记忆碎片会逐渐浮现。玩家最终会意识到，这个卧室并不是现实中的房间，而更像是主角反复经历的噩梦空间。

---

## 核心玩法

游戏采用任务阶段推进的方式。玩家需要根据环境线索、语音提示、字幕提示和物体发光提示，依次完成不同阶段的解谜任务。

主要玩法包括：

1. **探索卧室环境**
   玩家可以在 VR 中自由观察房间，通过头显和手柄感知空间中的物品位置。

2. **抓取与移动物体**
   玩家可以使用手柄抓取杯子、衣物、配件等物品，并将它们移动到指定位置。

3. **触碰与选择交互**
   玩家可以触碰电视、抽屉、钥匙、戒指等关键物体，触发剧情提示或任务推进。

4. **物品组合谜题**
   在整理衣物阶段，玩家需要将指定衣物和配件放置到正确区域，完成记忆恢复的一部分。

5. **顺序记忆谜题**
   在恢复记忆阶段，玩家需要按照叙事逻辑依次查看关键物品：

   * 戒指
   * 餐厅预订邀请函
   * 妻子的死亡通知单

   这一顺序对应“承诺——缺席——真相”的记忆推进逻辑。

6. **最终选择与多结局**
   当门被打开后，玩家将进入最终选择阶段。此时床和门都会成为可交互目标：

   * 走向门：象征逃避与离开
   * 回到床边：象征面对记忆与接受真相

   不同选择会进入不同结局演出。

## 交互说明

### VR 控制方式

* 使用手柄射线或直接触碰选择可交互物体
* 使用 Grab 按键抓取/触摸物品
* 移动手柄可以调整物体位置
* 靠近或触碰部分物体会触发剧情提示
* 关键阶段中，发光物体通常代表当前交互目标

### 视觉提示

游戏中使用发光系统提示关键物品：

* 短暂发光：提示该物体可被注意或刚刚触发过
* 阶段恒亮：表示该物体是当前任务目标
* 最终阶段中，床和门会同时发光，提示玩家进行选择

### 字幕与提示

游戏使用两类文字提示：

* 底部剧情字幕：用于表现妻子的声音、主角内心独白和剧情推进
* 右上角操作提示：用于提示玩家当前需要完成的操作

## 设计思路

本 Demo 的设计重点不是单纯完成谜题，而是让玩家通过交互主动参与叙事。

### 1. 用物品承载记忆

每个关键物品都代表一段记忆：

* 戒指代表承诺和亲密关系
* 餐厅邀请函代表被错过的约定
* 死亡通知单代表主角一直逃避的现实
* 床和门分别代表面对与逃避

玩家不是通过阅读大量文本理解故事，而是通过触碰、拾取和观察物品逐渐拼合真相。

### 2. 用任务阶段控制叙事节奏

游戏使用任务链系统推进流程。每个阶段只开放当前需要关注的目标，避免玩家在 VR 场景中迷失方向。同时，阶段之间通过字幕、音频和物体显隐进行衔接，使剧情推进更加自然。

### 3. 用 VR 交互增强沉浸感

VR 让玩家不只是观看故事，而是亲手执行动作。例如：

* 亲手拿起水杯
* 打开抽屉寻找照片
* 整理衣物
* 拾起戒指并看到它移动到视野前方后消失，模拟被戴上
* 在最后亲自走向门或床，完成选择

这些动作让叙事变成玩家身体参与的过程。

### 4. 用检查点系统保证体验连续性

由于 VR 中物体可能掉落或被玩家误操作，项目实现了检查点系统。任务完成后，关键物体的状态会被保存，包括：

* 物体是否显示
* 物体位置
* 物体旋转
* Rigidbody 物理状态
* 当前任务进度

这样玩家即使触发场景重置，也不会丢失已经完成的解谜进度。

## 系统功能概览

本项目包含以下主要系统：

* **XR 交互系统**：基于 XR Interaction Toolkit，实现抓取、选择、触碰等 VR 交互
* **任务链系统**：管理当前任务阶段、任务提示、阶段推进和任务完成逻辑
* **检查点系统**：保存关键物体状态和当前任务进度
* **物体发光系统**：用于提示当前关键物品
* **隐藏物体显现系统**：用于在指定阶段显示剧情物品
* **音频序列系统**：不同任务阶段可自动播放多段语音或环境音
* **字幕与提示系统**：显示剧情字幕和操作提示
* **最终选择系统**：控制门结局和床结局的触发
* **结局演出系统**：根据玩家选择播放不同结局图片、音频和字幕
